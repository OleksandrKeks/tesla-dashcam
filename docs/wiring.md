# Підключення в Tesla Model S 2017

## Схема

```
[Бардачок Tesla Model S]
┌──────────────────────────────────┐
│                                  │
│   USB-C порт (data + power)     │
│       │                         │
│       │ USB-A to USB-C кабель   │
│       │ (короткий, ~30см)       │
│       │                         │
│   [Raspberry Pi 4B]             │
│       │                         │
│       ├── USB-A: 4G Dongle      │
│       └── microSD: 128-256GB    │
│                                  │
│   [Корпус з пасивним            │
│    охолодженням]                │
│                                  │
└──────────────────────────────────┘
```

## USB підключення

### Model S 2017 з MCU2
- **Бардачок**: USB порт підтримує data transfer + Dashcam
- Використовуємо цей порт для RPi
- RPi живиться від цього ж порту (5V)

### Кабель
- USB-A (RPi) → USB-C (Tesla бардачок)
- Короткий (15-30 см) для акуратності
- Якісний кабель з підтримкою data transfer

## Живлення
- RPi живиться від USB порту Tesla (~1.2A доступно)
- 4G dongle живиться від RPi USB
- **Увага**: при Sentry Mode USB порт активний
- При глибокому сні Tesla USB вимикається

## Температурний режим
- Бардачок: помірна температура
- Корпус з "armor case" або пасивним радіатором
- Уникати прямого сонця (бардачок закритий — ОК)

## PIN-код бардачка
- Рекомендується встановити PIN на бардачок
- Controls → Glovebox PIN → увімкнути
- Захищає RPi від крадіжки

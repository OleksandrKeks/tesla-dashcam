# Схема підключення в Tesla Model S 2017

## Живлення

### Варіант 1: 12V розетка в бардачку
- 12V → DC-DC Step-Down (LM2596 або XL4015) → 5V 5A → Raspberry Pi
- Переваги: просте підключення, не потрібно різати проводку
- Мінуси: живлення тільки при увімкненому запаленні

### Варіант 2: Пряме підключення до 12V акумулятора
- Через запобіжник 10A → DC-DC → 5V
- Переваги: працює завжди (Sentry Mode)
- Мінуси: може розряджати 12V батарею
- Рішення: таймер відключення при напрузі < 11.8V

## Розміщення камер

### Передня камера
- За дзеркалом заднього виду
- USB кабель по стелі → до бардачку

### Задня камера
- Біля заднього скла (верхня частина)
- USB кабель по стелі та боковій стійці

### Бокові камери (ліва + права)
- На бокових дзеркалах або стійках B
- USB кабелі по порогах

## Розміщення Raspberry Pi
- **Бардачок** — оптимальна температура, захист PIN-кодом
- Або під сидінням водія

## Схема з'єднань

```
12V (бардачок)
    │
    ├── [Запобіжник 10A]
    │        │
    │   [DC-DC 12V→5V 5A]
    │        │
    │   [Raspberry Pi 4B] ── USB Hub (з живленням)
    │        │                 ├── Камера 1 (перед)
    │        │                 ├── Камера 2 (зад)
    │        │                 ├── Камера 3 (ліва)
    │        │                 ├── Камера 4 (права)
    │        │                 └── 4G Dongle
    │        │
    │        ├── I2C: MPU6050 (GPIO 2/3)
    │        └── microSD 256GB
    │
    └── GND
```
